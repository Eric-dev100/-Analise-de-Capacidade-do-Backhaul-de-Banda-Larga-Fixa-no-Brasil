# -Analise-de-Capacidade-do-Backhaul-de-Banda-Larga-Fixa-no-Brasil
Este notebook apresenta uma análise exploratória sobre a infraestrutura de banda larga fixa no Brasil, com foco na capacidade e taxa de ocupação das redes de distribuição (backhaul). O objetivo principal é mapear a saúde da infraestrutura de telecomunicações através das regiões do país, identificando possíveis gargalos de rede.
